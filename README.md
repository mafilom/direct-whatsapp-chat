# Direct Chat Without Save Number

## 🚀 About
This is a lightweight, high-performance web tool that allows users to start a WhatsApp chat without saving a phone number. Designed with a modern Neumorphism (Soft UI) interface, it delivers a smooth and clean user experience.

Perfect for freelancers, businesses, and individuals who frequently need to contact new numbers without cluttering their contact list.

---

## ✨ Key Features

- Instant WhatsApp chat without saving contacts
- Clean Neumorphic (Soft UI) design
- Fully responsive (mobile-first)
- Optional pre-filled message support
- No data storage or tracking
- Fast and lightweight (ideal for GitHub Pages)

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Neumorphism Design)
- Vanilla JavaScript

## 📌 Usage

1. Enter phone number with country code
2. Add optional message
3. Click "Start Chat"
4. Redirect to WhatsApp instantly
